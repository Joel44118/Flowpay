# FlowPay
A simple payment page built with HTML, CSS, and JavaScript.

## Features
* A simple payment form with input fields for amount, card number, expiration date, and security code.
* Form validation to ensure all fields are filled in.
* A simulated payment processing delay.

## Usage
1. Open the `index.html` file in a web browser.
2. Fill in the payment form and submit it.
3. The payment will be processed and a success message will be displayed after a short delay.
